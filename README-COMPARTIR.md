# 📚 Flashcard Study App - Guía de Instalación

## 🎯 ¿Cómo compartir esta aplicación?

### Opción 1: Carpeta Completa (Recomendado)
1. **Comprime esta carpeta completa** en un archivo .zip
2. **Envía el .zip** a tus amigos
3. **Instrucciones para ellos:**
   - Descomprimir el archivo
   - Abrir terminal/cmd en la carpeta
   - Ejecutar: `npm install`
   - Ejecutar: `npm run dev`
   - Abrir `http://localhost:5173`

### Opción 2: Solo Archivos Build (Para usuarios avanzados)
1. Comprime solo la carpeta `dist/`
2. Los amigos necesitan:
   - Un servidor web local (como Live Server en VS Code)
   - O subirlo a un hosting gratuito

### Opción 3: GitHub (La mejor opción)
1. Sube este proyecto a GitHub
2. Los amigos pueden:
   - Clonar el repositorio
   - Seguir los pasos de la Opción 1

## 🔧 Requisitos del Sistema
- Node.js 16+ instalado
- NPM (viene con Node.js)

## 📱 Características
- ✅ Crear y gestionar decks de flashcards
- ✅ Añadir preguntas y respuestas
- ✅ Modo de estudio con animación 3D flip
- ✅ Persistencia de datos (localStorage)
- ✅ Diseño responsive para móviles
- ✅ Sin necesidad de base de datos

## 🎮 Uso Rápido
1. Crea tu primer deck
2. Añade tarjetas con preguntas/respuestas
3. Usa el modo de estudio para practicar
4. ¡Mejora tu aprendizaje!
